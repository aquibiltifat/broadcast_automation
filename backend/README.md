# Group Weaver AI Backend

## Setup
1. Copy `.env.example` to `.env`
2. Add your Google AI Studio API key to `.env`
3. Run `npm install`
4. Run `npm run dev`

## API Endpoints
- `POST /api/analyze` - Analyze broadcast lists for common members
- `POST /api/suggest-name` - Get AI-suggested list name
- `GET /api/health` - Health check
